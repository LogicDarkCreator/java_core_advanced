# Financial Accounting Application

Консольное приложение для учета доходов и расходов с историей транзакций

## Возможности

- Добавление транзакций (доходы / расходы)
- Просмотр последных 5 транзакций
- Формирование финансового отчета
- Расчет баланса
- Очистка всех транзакций

## Требования
- Java 17 или выше
- Maven 3.6 или выше
